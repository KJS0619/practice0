# 네이버 블로그 SEO 가이드

네이버 블로그 노출/검색 알고리즘의 최신 기준(C-Rank, D.I.A.+, 스마트블록, AEO/GEO)을 반영한 작성 가이드입니다.

---

## 1. 알고리즘 3종 이해

### C-Rank — 블로그(채널) 단위 신뢰도 평가
블로그 전체의 신뢰도·전문성을 평가하는 지표. 아래 3가지 요소로 구성됩니다.

| 요소 | 의미 | 가중치 |
|---|---|---|
| Context | 주제 집중도 (특정 주제를 얼마나 일관되게 다루는가) | 전문성 40% |
| Content | 콘텐츠 품질 (문서 자체의 충실도) | 지속성 30% |
| Chain | 콘텐츠의 공유/인용 확산도 | 반응 20% |

→ **블로그를 잡블로그처럼 운영하지 말고, 특정 주제(카테고리)에 집중해서 꾸준히 발행**해야 C-Rank가 쌓입니다.

### D.I.A.+ — 문서(포스트) 단위 품질 평가
개별 글 하나하나의 품질을 딥러닝 기반으로 평가하는 알고리즘.

파이프라인: **검색 의도 파악 → 문서 구조 이해 → 동적 랭킹**

→ 키워드 매칭만으로는 부족하고, **검색자의 의도에 맞는 정보가 잘 구조화되어 담겨 있는지**가 랭킹에 직접 반영됩니다.

### 스마트블록 (2025년~)
- VIEW 탭에 완전 통합되어, 이제 블로그/카페 글이 스마트블록 단위로 노출됩니다.
- AI가 검색 의도를 분석한 뒤, 문서를 통째로 노출하지 않고 **블록(문단) 단위로 재구성해서 보여줌**.
- → 문서 전체가 아니라 **문단 하나하나가 독립적으로 평가받는다**고 생각하고 작성해야 합니다. (문단마다 하나의 소주제 + 명확한 결론)

---

## 2. 2026 신패러다임 — AEO + GEO

- **AEO (Answer Engine Optimization)**: 검색 즉시 정답을 보여주는 방식에 최적화. 질문 → 바로 답이 나오는 구조가 유리.
- **GEO (Generative Engine Optimization)**: 생성형 AI(검색 요약, 챗봇 등)가 내용을 인용하기 좋은 형태로 최적화.
- 핵심 변화: **"사람이 읽기 전에 AI가 먼저 해석한다."**
  → 감성적 나열형 글보다 **정리된 정보 문서** 형태(소제목, 목록, 요약 문단)로 작성하는 것이 필수가 되었습니다.

## 3. 작성 규칙

### 제목 규칙
- 메인 키워드는 **1~2회**만 포함 (반복 남발 금지)
- 메인 키워드는 **제목 앞쪽**에 배치
- **숫자 또는 혜택**을 포함해 클릭을 유도 (예: "3가지 방법", "50% 절약하는 법")

### 본문 구조
- 전체 글 길이: **1,500~2,000자**
- 스마트블록 대응을 위해 **소제목 단위로 문단을 분리**하고, 각 문단은 하나의 소주제 + 결론으로 마무리
- 서두에 검색 의도(질문)를 짚어주고, 본문에서 답을 주는 **AEO형 구조**를 우선 고려

### 키워드 배치
- 제목 + 본문 합산 **5~6개** 키워드 사용
- 특정 문단에 몰지 말고 **글 전체에 자연스럽게 분산**
- 문맥에 맞게 자연스럽게 녹여 쓰기 (반복 삽입 금지)

### 이미지
- 개수: 해당 키워드 **상위 1~5위 글의 평균 이미지 개수**에 맞춤
- **직접 촬영한 이미지 우선** 사용
- 타 블로그/웹과 **유사·중복 이미지 사용 금지**

### 해시태그
- **10~20개** 사용
- 포괄적인 태그보다 **구체적인 태그** 위주로 선정
- **글 하단**에 배치

---

## 4. 하지 말 것 (금지사항)

- 키워드 도배 (동일 키워드 반복 삽입)
- 타 블로그/문서 복붙
- AI가 생성한 글을 그대로 검수 없이 발행
- 숨김 텍스트 (흰 글씨, 사이즈 0 등)
- 댓글/공감 매크로 사용
- 낚시성 제목 (내용과 무관한 자극적 제목)
- 쇼핑/제휴 링크 도배
- 주제 없는 잡블로그 운영 (C-Rank Context 점수 저하)
- 다중 IP를 이용한 어뷰징

## 요약 체크리스트

- [ ] 제목에 메인 키워드 1~2회, 앞쪽 배치 + 숫자/혜택 포함
- [ ] 본문 1,500~2,000자, 소제목 단위로 문단 분리
- [ ] 키워드 5~6개 자연스럽게 분산
- [ ] 이미지는 상위 글 평균 개수, 직접 촬영, 중복 없음
- [ ] 해시태그 10~20개, 구체적 태그, 하단 배치
- [ ] 금지사항(도배·복붙·매크로·낚시제목 등) 미포함 확인
